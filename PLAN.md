# MarkFlow - Web Markdown Editor

## 1. 项目概述

**项目名称**：MarkFlow
**技术栈**：React + TypeScript + CodeMirror 6 + Vite
**定位**：Web 版 Typora，即时渲染的 Markdown 编辑器

---

## 2. 核心特性

### 2.1 编辑功能
| 特性 | 描述 | 状态 |
|-----|------|------|
| 即时渲染 | 边写边显示渲染结果 (300ms) | ✅ 已完成 |
| 分屏预览 | 左右分栏 | ✅ 已完成 |
| 键盘快捷键 | Ctrl+B/I/K | ✅ 已完成 |

### 2.2 Markdown 支持
- CommonMark + GFM ✅
- 代码高亮 ✅
- 数学公式 (KaTeX) ✅
- 表格 ✅
- 任务列表 ✅

### 2.3 主题
- 浅色/深色主题 ✅
