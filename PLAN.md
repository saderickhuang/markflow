# MarkFlow - Web Markdown Editor

## 1. 项目概述

**项目名称**：MarkFlow
**技术栈**：React + TypeScript + CodeMirror 6 + Vite
**定位**：Web 版 Typora，即时渲染的 Markdown 编辑器

---

## 2. 核心特性

### 2.1 编辑功能
| 特性 | 描述 | 优先级 |
|-----|------|--------|
| 即时渲染 | 边写边显示渲染结果 | P0 |
| 分屏预览 | 左右/上下分栏 | P0 |
| 源码模式 | 纯文本编辑 | P1 |
| 专注模式 | 隐藏 UI | P2 |

### 2.2 Markdown 支持
- CommonMark + GFM
- 代码高亮（多语言）
- 数学公式（KaTeX）
- 表格
- 任务列表
- 目录自动生成

### 2.3 文件操作
- 新建/打开/保存
- 拖拽打开
- 自动保存
- 导出 PDF/HTML

### 2.4 主题
- 浅色/深色主题
- 主题切换

---

## 3. 技术架构

```
┌─────────────────────────────────────────┐
│              React + TypeScript          │
├─────────────────────────────────────────┤
│              UI Components              │
│  - EditorPane (CodeMirror)             │
│  - PreviewPane (react-markdown)        │
│  - Toolbar                             │
│  - StatusBar                           │
├─────────────────────────────────────────┤
│              Services                   │
│  - MarkdownService (解析)              │
│  - FileService (文件操作)              │
│  - ThemeService (主题)                │
│  - ExportService (导出)               │
├─────────────────────────────────────────┤
│              Core Libs                  │
│  - CodeMirror 6 (编辑)                │
│  - react-markdown (渲染)              │
│  - KaTeX (数学公式)                   │
│  - Vite (构建)                        │
└─────────────────────────────────────────┘
```
